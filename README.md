# bingEm
Basic structure of app set up with express-generator using:
* node.js
* express server
* ejs templates
* css support - scss

## Directory structure

```
|-README.md
|-app.js
|
|-/public/
|    |-css/
|    |-js/
|    |-images/
|    |-javascripts/
|    |-stylesheets/
|
|-/routes/
|    |-index.js
|
|-/views/
     |-index.ejs
```

## Commands
Basic commands you can run with the app.

Compile Scss and start App: `npm run start`

Compile scss: `npm run sass`
